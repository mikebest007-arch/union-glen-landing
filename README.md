# Union Glen Landing Page

## Overview
Stunning full-screen landing page for Union Glen townhomes with hero rendering, townhome collections, and builder showcase.

## Files
- `index.html` - Main landing page
- `hero.jpg` - Main hero rendering (Deco elevation)
- `logo.jpg` - Union Glen logo
- `classic.jpg` - Classic Towns rendering
- `tandem.jpg` - Tandem Garage rendering
- `double.jpg` - Double Garage rendering
- `courtyard.jpg` - Courtyard Towns rendering
- `builders.jpg` - All 7 builder logos

## Deployment

### Option 1: Deploy to unionglen2026.com
Upload all files to your web server root.

### Option 2: Use as subdirectory
Upload to `/landing-page/` folder and access via `unionglen2026.com/landing-page/`

### Option 3: Replace existing site
Backup current site, then upload these files to replace it.
